# MERN Authentication Backend

## Features
- User Registration
- User Login
- JWT Authentication
- Protected Routes
- MongoDB Integration
- Environment Variables
- Error Handling

## Installation
npm install

## Run
npm run dev

## Environment Variables
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=_jwt_secret_2026

## API Endpoints
POST /api/auth/register
POST /api/auth/login
GET /api/auth/profile
